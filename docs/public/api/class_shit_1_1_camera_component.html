详细描述

相机组件，定义"从哪个角度观察世界"

相机的位置由同 GameObject 上的 TransformComponent 决定。 m_worldSize 决定能看到的世界范围，m_viewportRatio 决定渲染到屏幕的哪个区域。 支持多相机分屏渲染（按 priority 排序）。
