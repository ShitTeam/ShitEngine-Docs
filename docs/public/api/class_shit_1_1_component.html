详细描述

组件基类

所有组件必须继承此类。

生命周期（按调用顺序）： onCreate — 组件刚被构造、已绑定 owner，尚未挂载到场景 onAttach — 组件所属 GameObject 进入活动场景时 onDetach — 组件即将从场景中移除时（在 onDestroy 前调用） onDestroy — 组件被销毁时
