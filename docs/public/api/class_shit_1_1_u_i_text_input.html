友元

class TextInputGate

详细描述

文本输入控件抽象基类

负责所有输入控件共享的状态与行为：

  • 文本缓冲（UTF-8）+ 光标位置（以 UTF-8 字符计）
  • 选区（Shift + 方向键 / 鼠标拖拽）
  • 占位符文字（无内容且未聚焦时显示）
  • IME 组合串（preedit）展示与插入
  • 聚焦状态，失焦自动停止文本输入模式

单行 vs 多行的差异由子类覆写 onRender / onKeyDown / 插入换行策略实现。 不直接绘制背景：背景由同 GameObject 上的 UIImage 提供（与按钮一致）。
