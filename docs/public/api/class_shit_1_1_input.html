输入管理（单例）

同时提供两层 API（类似 Unity / Godot）：

  1. 底层键鼠 Input::IsKeyDown(KeyCode::Space) Input::IsMouseButtonPressed(MouseButton::Left)
  2. 动作 / 轴映射（配置在 settings.json 的 inputMappings） Input::IsActionDown("Jump") Input::IsActionPressed("Jump") Input::IsActionReleased("Jump") Input::GetAxis("Horizontal") // -1 / 0 / 1

命名约定（与 Unity 不同，请注意）： Down = 刚按下（仅一帧） Pressed = 持续按住 Released = 刚松开（仅一帧）

键名使用 SDL 官方 scancode 名（"Space"/"A"/"Left Shift"）， 也接受无空格别名（"LeftShift" → "Left Shift"）。 鼠标：MouseButton.Left / Right / Middle / XButton1 / XButton2
