详细描述

输入管理类（单例）

提供键盘鼠标的三态检测（Down / Pressed / Released）和鼠标位置查询。 所有 API 均为静态方法。

命名提醒： IsKeyDown = 刚按下（瞬间） IsKeyPressed = 持续按住（每帧 true 直至松开） IsKeyReleased= 松开（瞬间）
