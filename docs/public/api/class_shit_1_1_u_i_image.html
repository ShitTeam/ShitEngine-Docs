详细描述

UI 图片控件，在屏幕空间绘制一张精灵

继承 UIRendererComponent，由 UIRenderSystem 每帧调用 onRender。 位置 / 尺寸由同 GameObject 上的 UITransform 决定；Color 用于按钮状态着色。 不走游戏世界的相机变换，直接用屏幕坐标。
