详细描述

UI 渲染控件基类（对应游戏世界的 RendererComponent）

所有可渲染的 UI 控件（UIImage / UIText / UIButton 等）继承此类。 onAttach / onDetach 自动向 UIRenderSystem 注册 / 注销。 由 UIRenderSystem 每帧按 zIndex 排序后统一调用 onRender，屏幕矩形由 UITransform 计算。
