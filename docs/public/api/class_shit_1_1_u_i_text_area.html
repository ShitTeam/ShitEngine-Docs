详细描述

多行文本输入区域

支持换行（Enter 插入 '
'），自动换行由后续版本支持；当前为软包裹可选。 上下方向键按"逻辑行"移动光标（按 '
' 切分文本）。

用法与 UITextBox 一致，构造即可用。
