场景管理器（单例）

使用栈结构管理场景生命周期。任何时候只有栈顶场景处于活跃状态。 所有操作延迟到 Update() 时执行，保证迭代安全。

用法： SceneManager::PushScene(std::move(myScene)); SceneManager::ReplaceScene(std::move(gameScene)); SceneManager::PopScene(); SceneManager::ClearScene();
