详细描述

行为基类 —— 用于编写自定义游戏逻辑

继承自 Component，扩展出 onStart / onUpdate 两个阶段： onCreate → onAttach → onStart → (每帧)onUpdate → onDetach → onDestroy

onStart 在首次 update 前执行一次，适用于缓存指针。 onUpdate 每帧执行，适用于输入、移动、碰撞检测等。

挂载后由 BehaviorSystem 自动驱动，无需手动调用。
