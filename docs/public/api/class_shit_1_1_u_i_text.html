详细描述

UI 文字控件，使用 SDL_ttf 渲染文字到纹理

纹理缓存策略：setText / setFontSize / setColor 置 dirty， 下次 onRender 用 TTF_RenderText_Blended 重新生成 SDL_Texture。 onDestroy 释放缓存的纹理。字体按 path+size 加载并缓存于进程级映射。 对齐方式（左 / 中 / 右）相对 UITransform 的屏幕矩形。
