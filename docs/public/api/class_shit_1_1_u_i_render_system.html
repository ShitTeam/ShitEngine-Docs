详细描述

UI 渲染系统

每帧执行两件事：

  1. 输入 Raycasting：按 zIndex 从上到下检测鼠标命中的 UI 元素，驱动其上的 UIButton 状态；
  2. UI 渲染：按 zIndex 从下到上调用各 UIRendererComponent 的 onRender，叠在游戏世界之上。

UIRendererComponent 与 UICanvas 的 onAttach / onDetach 自动注册 / 注销。 priority 默认 200，晚于 RenderSystem(100)，确保 UI 绘制在游戏世界之后。
